inode

index + node

inode就是用来维护某个文件被分成几块、每一块在的地址、文件拥有者，创建时间，权限，大小等信息。

Linux常用指令

指令 作用
stat 查看文件的 inode 信息
ln 创建硬链接
file 查看文件类型信息
tar 压缩和解压缩
ls -l 查看某个目录下的文件或目录的权限
top 查看系统的 CPU 使用率、内存使用率、进程信息等
systemctl 查看系统服务的状态、启动、停止、重启等

Linux获取实时刷新的日志

linux
说几个基本linux命令
1.cd 切换目录，2.mkdir创建目录,3.mv移动或重命名 ,4.cp复制, 5.touch 创建文件 6.vi编辑器，创建文件，7.find查找8.cat/tail/head/less查看 9.ps-ef查看进程 10.ifconfig 查看ip地址 11.rm删除
file
file通过探测文件内容判断文件类型，使用权限是所有用户
格式file[options]文件名
[options]主要参数
-v：在标准输出后显示版本信息
-z：探测压缩过的文件类型
-L：允许符合连接
mkdir
mkdir命令的作用是建立名称为dirname的子目录，与MS DOS下的md命令类似，它的使用权限是所有用户
格式：mkdir[options]目录名
grep
grep命令可以指定文件中搜索特定内容，并将含有这些内容的行标准输出。Global Regular Expression Print，表示全局正则表达式版本，它的使用权限是所有用户。该命令常用于分析一行的信息，若当中有我们所需要的信息，就将该行显示出来，该命令通常与管道命令一起使用，用于对一些命令的输出进行筛选加工。
pgrep的p表明了这个命令是专门用于进程查询的grep
find
find命令的作用是在目录中搜索文件，它的使用权限是所有用户
格式：find[path][options][expression]path 指定目录路径，系统从这里开始沿着目录树向下查找文件，它是一个路径列表，相互用空格分离，如果不写path，那么默认为当前目录
ls：查看当前文件与目录
cd：用于切换当前目录
cd ~：当前用户的家目录下的路径信息
cd -：切换到上一次所在的路径
cp：复制命令，在同一个linux系统上，不同的目录间复制文件
cp -i：交互式复制，覆盖目标文件会给出提示
cp -r：若要复制的文件是目录文件，则将递归复制目录下的全部子目录和文件，
scp：在不同的linux系统之间来回复制文件
mv：移动或重命名文件或文件夹命令（如果目标文件名不是文件夹或文件名不存在，则为修改名称）
mv -i：交互式复制，覆盖目标文件会给出提示
rm：删除文件或文件夹命令
-i：交互式
-r：递归，删除文件夹必须加此参数
touch：创建文件
mkdir：用于创建新目录(文件夹)
ps：查看进程情况
kill：向进程发送终止信号
$ kill -s 9 1827，1827是进程号，由ps -aux查看（或ps -ef），其中-s 9 制定了传递给进程的信号是９，即强制、尽快终止进程
pwd：用于显示工作目录
tar：对文件进行打包，调用gzip或bzip对文件进行压缩或解压
cat：查看文件内容，与less、more功能类似
top：可以查看操作系统的信息，如进程、cpu占用率、内存信息等
查看进程：ps -ef|grep 进程id 【ps:将某个进程显示出来 -A 　显示所有程序。 -e 　此参数的效果和指定”A”参数相同。 -f 　显示UID,PPIP,C与STIME栏位。】
ps命令： ps命令查找与进程相关的PID号：
ps a 显示现行终端机下的所有程序，包括其他用户的程序。
ps -e//ps -A 显示所有程序。
ps u 以用户为主的格式来显示程序状况。
ps x 显示所有程序，不以终端机来区分。
【ps aux】
ps c 列出程序时，显示每个程序真正的指令名称，而不包含路径，参数或常驻服务的标示。
ps -e 此参数的效果和指定”A”参数相同。 ps e 列出程序时，显示每个程序所使用的环境变量。
ps f 用ASCII字符显示树状结构，表达程序间的相互关系。
杀死叫firefox的进程：ps aux| grep firefox | awk ‘{print $2}’ |xargs kill *xargs 命令是用来把前面命令的输出结果(PID)作为“kill -s 9”命令的参数，并执行该命令
$pkill -９ firefox *说明：pkill＝pgrep+kill，”-9” 即发送的信号是9，pkill与kill在这点的差别是：pkill无须 “ｓ”，终止信号等级直接跟在 “-“ 后面
pgrep的p表明了这个命令是专门用于进程查询的grep
查看端口号: netstat –tunlp|grep 端口号 lsof -i:端口号
如果想把一个文件移动到另一个文件夹里面去，用什么命令
windows move [源文件路径] [目标文件夹路径]
move D:\test.txt D:\backup
Linux或MacOS系统下，可以使用mv命令来移动文件 mv [源文件路径] [目标文件夹路径]，mv命令既可以用来移动文件，也可以用来重命名文件
mv /home/user/test.txt /home/user/backup/
如何查看Linux系统进程？
通过ps命令可以查看当前系统运行的进程的列表，包括PID，进程名，CPU占用百分比，内存占用等信息。
可以使用top命令来查看Linux系统正在运行的进程。
如何查看a.txt的第7到第9行？
查看文件a.txt的第7-9行
cat a.txt | tail -n +7 | head -n 9
查看文件a.txt的前3行
cat a.txt | head -n 3
查看文件a.txt，显示第50行到第200行
cat a.txt | head -n 200 | tail -n +50
查看a.txt文件的第5行
head -n 5 a.txt | tail -n 1
一个文件中字段以逗号隔开，如何查看某一列的所有的数据？
awk -F[,] ‘{print $10}’
如何查看程序运行时的某个端口？
lsof -i :<端口号>
给文件夹或者文件增加权限
chmod +777
